# DrivePulse Web V2

Nouveautés : quatre boucles WAV synchronisées, mixage adaptatif, réglage manuel des couches et mode démonstration.

## Mise à jour GitHub Pages
Remplace les fichiers de la V1 par ceux-ci et conserve le dossier audio. Recharge ensuite le site dans Safari. Si l’ancienne version reste affichée, supprime l’icône de l’écran d’accueil, recharge Safari, puis ajoute-la de nouveau.

## Test
Lance « Charger et démarrer », puis « Mode démonstration ». Teste ensuite les quatre curseurs avant l’essai en voiture.
