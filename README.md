# DrivePulse Web V5.2

## Piano à l’arrêt
Un nouveau bouton permet d’activer ou de désactiver un solo de piano lorsque la voiture est quasiment à l’arrêt.

Le solo est une composition originale fondée sur des codes généraux de :
- progressive house ;
- piano EDM mélodique ;
- musique électronique émotionnelle ;
- alternance entre notes graves, accords et réponses aiguës.

Il ne reprend pas la mélodie de `KDrew - Circles` et n’imite pas directement un artiste précis.

## Fonctionnement
- Activé : solo de piano plus présent sous 8 km/h.
- Désactivé : fond synthétique plus discret.
- Dès que la voiture accélère, les couches dynamiques reprennent progressivement le dessus.
