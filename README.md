# DrivePulse Web V8.2 — Piano basse vitesse & stem transparency

La V8.2 conserve le moteur audio et la fusion de capteurs de la V8.1, et ajoute un stem Rhodes dédié.

## Nouveautés

- bus `PIANO` séparé ;
- piano actif de l’arrêt jusqu’à 30 km/h ;
- fondu progressif entre 26 et 34 km/h pour éviter une coupure brutale ;
- réduction automatique du bus Harmony lorsque le piano est présent ;
- conditions d’activation affichées sous chaque personnage ;
- documentation complète dans `STEM_BEHAVIOR.md` ;
- mapping mis à jour dans `STEM_MAPPING.csv`.

## Source musicale

`Kalte Ohren (Cold Ears)` — starfrosch feat. Jerry Spoon, CC BY 3.0.

## Installation GitHub

Créer la branche `drivepulse-v8-2`, puis remplacer l’intégralité de la V8.1.

Commit conseillé :

`DrivePulse V8.2 - low speed piano and documented stem behavior`
