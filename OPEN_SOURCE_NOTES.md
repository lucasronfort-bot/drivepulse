# Open-source and copyright notes

## Incredibox

No Incredibox code, audio, visual asset, melody, character or arrangement is included in DrivePulse.

## Vocal synthesis

The V7.3 voice engine is an original Web Audio API implementation using formant-filter synthesis.

Its general technical direction is informed by open articulatory-synthesis research and projects such as Pink Trombone, but no Pink Trombone source file is copied or bundled in this release.
